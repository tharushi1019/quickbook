# QuickBook

A simple service booking system built with PHP, MySQL, HTML, CSS, and JavaScript.

## Features
- Service booking form
- Admin dashboard with authentication
- Approve / Delete bookings
- Confirmation dialogs & success messages
- Responsive UI

## Tech Stack
- PHP
- MySQL
- HTML, CSS, JavaScript
- XAMPP

## How to Run
1. Clone the repository
2. Place it in `htdocs`
3. Import database
4. Start Apache & MySQL
